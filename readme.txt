To run dashboard locally do the following 
- Install shiny and python version 3.11.6
- Install the following packages : 
	pandas
	numpy
	matplotlib
	seaborn
	shiny
	shinywidgets
	ipyleaflet
	ipywidgets
	plotly
	shiny[theme]
- Collision.csv and guide.xlsx will have to be in the same folder as the dashboard
- Run using the command "python -m shiny run "(local directory)\collision_dashboard.py""
